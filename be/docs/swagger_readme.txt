Install and Configure Swagger

1. Run command in backend directory:
$ npm install swagger-jsdoc swagger-ui-express --save-dev

2. add swagger.js file to root backend directory (and adjust the content to your liking)

3. modify main.js with the one included in this folder, mainly to import swagger.js
and call swaggerDocs function.


Useful links:
https://swagger.io/docs/specification/about/
https://swagger.io/docs/specification/authentication/basic-authentication/
